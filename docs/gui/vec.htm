1. To change the vector threshold, move the Vector Threshold Max% and Min% sliders under Vector Controls.

2. To change the vector scale, move the vector scale slider under Vector Controls.

3. To change the vector ratio, move the fector ratio slider under Vector Controls.

4. To scale by vector mag, check Scale by Vector Mag under Vector Controls.

5. To change contour opacity, move the Contour Opacity slider under Contour Controls.

6. To change the warped contour scale, move the Warped Contour Scale under Contour Controls.

7. To change the contour LOD, move the Contour LOD slider under Contour Controls.
