1. To change cursor selection, select one of the options below Cursor Selection (none, point, line, plane).

2. To change the axis of direction, select one of the options below Direction (X, Y, Z).

3. To change the integration direction, select one of the options below Integration Direction (backward, forward, both directions).

4. To change the propagation time, move the Propogation Time slider below Streamline Controls.

5. To change the integration step, move the Integration Step slider below Streamline Controls.

6. To change the step, move the Step slider below Streamline Controls.

7. To change the size, move the Size slider below Streamline Controls.

8. To change the number of points, move the Number of Points slider below Streamline Controls.

9. To change the line diameter, move the Line Diameter slider below Streamline Controls.
