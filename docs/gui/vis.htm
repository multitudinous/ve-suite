1. To change the category, choose an option below Category (contour, warped contour, vector, isosurface, PIV_Image, animated contours, polydata).

2. To change the contour type, choose an option below Contour Type (graduated, banded, lined).

3. To change the direction, choose an option below Direction (X, Y, Z, By wand).

4. To change the type, choose an option below Type (All precomputed serfaces, Specify a single pane).

5. To make transient, check the Transient checkbox.

6. To use scalar bar, check the Scalar Bar checkbox.

7. To record the scene, click Record Scene.

8. To clear all, click Clear All.

9. To exit, click Exit.

10. To activate custom visualizations, click Activate Custom Visualization.
