Using VE-Builder with StarCD

(Collecting and Saving Data from StarCD)

This section will instruct the user how to store certain datasets from a StarCD *.pst file for use later.

1. Launch StarCD.

2. Ensure that a *.vrt and *.cel file is written of the region of interest. This region can either be a complete CFD model or a subsection of the model.

3. Ensure there is a *.pst file with all the data.

Have the Star GUIde panel open.

4. In the Star GUIde, click on the folder "Post-processing > Load data".

4.1 In the blue "File(s)" tab, click "Open Post File".

Initially, the velocity components and the Turbulent Kinetic Energy will be stored in the Post Registers. This is done by the following.

4.2 In the blue "Data" tab:

--Scalar Data > Turb Kinetic Energy.

--Vector Data > Velocity Components UVW.

5. Click "Get Data".

This data is now stored in the Post Registers.

6. To view currently stored data, Click "View Post Registers".

The left column lists the vertex number and the subsequent columns list the specified data for the respective vertices.

Please note: The Post Register stores the VERTEX data that was selected from the "Data" tab, the Turbulent Kinetic Energy, and the three components of the Velocity.

Up to six parameters of vertex information can be stored in the post registers. The added data used for this example is Pressure and Viscosity.

To add other scalar data:

7.1 Click "Operate".

7.2 Function > Load vertex data > turbulent energy (Registers 1-3 must contain vector data. Registers 4-6 must contain scalar data.)

7.3 Select register.

Note: Although it is necessary to use vertex data rather than cell data, using vertex data might return an error. To solve this problem, copy the previous command line and add the register number, e.g. "enter oper, getv, p, 5,relative "

7.4 Ensure that "using relative values" is selected.

7.5 Click "Apply" to put data in an empty register.

7.6 Click "Update list".

7.7 Repeat steps 7.1-7.6 to add data to register 6 (viscosity).

8. The data is now stored.

Now the data should be saved.

In the pro-STAR window:

9. File > save as coded.

10. Unclick boundary file.

11. Click "Apply".

Note: Checked files are written out.

12. Enter "close all".

You now have the geometry information and data. It is time to translate.
