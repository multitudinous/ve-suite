SOFC 1D Module

The SOFC_1D model is of higher fidelity compared to the 0-D SOFC model also in the workbench. The model allows for steady-state spatial variation in one dimension along the fuel cell.

Connections

  • Inlet Port 0 (left): Inlet anode gas stream (rich in CO and H2).
  • Inlet Port 1 (top): Inlet cathode gas stream.
  • Outlet Port 0: Outlet gas stream.

Graphical User Interface (UI)

This user interface allows the user to manipulate fuel cell characteristics and perfomance. The user can input not only fuel cell size and quantity, but also the cell resistance and general construction of the unit.


Summary Data (SUM)

This screen shows results after the module is executed. The first line shows overall produced power and heat transfer of the unit.
