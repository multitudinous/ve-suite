Compress_Expand Module Buttons

Graphical User Interface (UI)

The User Interface first allows the user to input an Isentropic Efficiency for the Compressor/Expander (as fraction).

The user must also specify a desired Pressure Change. Negative values decrease pressure through the module--modeling an Expander. Positive values increase pressure--modeling a Compressor.

Summary Data (SUM)

The summary table indicates the energy required (for compression), or energy generated (during expansion), as a result of module execution.
