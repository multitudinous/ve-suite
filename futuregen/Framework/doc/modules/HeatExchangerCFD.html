Heat Exchanger CFD Module

The Heat Exchanger CFD Module utilizes the Glacier CFD code. It is an experiment module. It is use to demonstrate the CFD to CFD coupling capability of the framework. Do not use this module to replace Heat Exchanger Module in ordinary simulation.


Connections

  • Inlet Port 0 : Inlet gas stream
  • Outlet Port 0 : Outlet gas stream



Summary Data (SUM)

 After the module has executed, the Summary Data button provides summary information. The module outputs calculated soot mass fraction, soot particle #/mass fraction and tar mass fraction.
