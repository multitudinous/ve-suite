Building/Configuring a Network

Beginning a Network

 

Adding a Module

To begin construction of a module network, first select a module from one of the module folders (Gas, Components, Water) located in the Module frame in the top left corner of the screen under the folders Available Modules and REI. Select a module by placing the cursor over the module and double-clicking the left mouse button.  The selected module will appear within the NetEdit frame.  To move the module within the NetEdit frame, place cursor over module, depress left mouse button, move module to desired location, and release left mouse button. 

 

 

Port Connectors

To connect two modules, place the cursor over the outlet port of one module, depress the left mouse button and drag the mouse to the inlet port of the second module.  Upon release, the two modules will be linked with an arrow pointing the direction of data flow.  Note that a module’s inlet and outlet ports appear only when the cursor is placed over the module.

 

 

 

 

To add a link connector, place cursor on a port connector and click right mouse button.  From the menu of options that appears, select Add Link Connector.  Three green boxes will be displayed at the inlet and outlet ports of the modules and in the center of the port connector.  Place cursor on the green box in the center of the port connector, depress the left mouse button, drag the green box to desired position and then release left mouse   button. 

 

 

 

 


Configuring the Modules

Once all modules within the network are correctly placed and connected, configure each module by placing the cursor over a module and double-clicking the left mouse button.  The Module Specification window will be displayed which contains unit characteristics such as initial conditions, unit geometry, and process specifications.  Within this window, complete the required fields.

 

 

 

 

 

 


Removing Modules and Connections

Placing the cursor over a module or port connector and selecting the right mouse button summons a menu of options. To remove a module, port connector, or link connector from the NetEdit frame, select Del Module, Delete Link, or Delete Link Connector from this menu of options. Note that all existing connections to deleted modules are automatically removed.
