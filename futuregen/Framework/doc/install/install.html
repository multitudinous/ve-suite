SCIRun Installation and Setup

SCIRun, the primary interface for use of the Vision 21 Workbench, is developed by the Scientific Computing and Imaging Institute. For detailed documentation on how to install the SCIRun software please refer to the SCIRun Installation Guide.

As illustrated on the SCIRun page, including the Vision 21 and REI Module Package requires an additional line when configuring SCIRun. To install REI's Vision21 Workbench simply add: 

 
--enable-package="REI Vision21"

to the configure command line.
