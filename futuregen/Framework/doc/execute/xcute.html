Executing a Module Network

Executing a Configured Module Network

In order to execute a configured modules network, first select the UI Module Button on a data source module. From this User Interface menu (shown) select the Execute button. All data source modules (Gas Source or Water Source modules) in the network will thereby simultaneously initiate data flow.

Right-clicking on any module and selecting Execute from the menu also performs the same operation.

Network Progress

A module's progress bar turns green to signify that the module has completed its computations. The number immediately to the left of the progress bar is the CPU time that was required to complete computations for that module.

When complicated calculations require extended time, a row of question marks appear immediately to the left of the progress bar to signify that calculations continue within the module.
