Executing a Module Network

Executing a Configured Module Network

In order to execute a configured modules network, select Connection located on the task menu bar and Connect Executive from the menu of options that is made available. Once the statement, “Connected to Executive” is shown in the Error frame, select Connection from the task menu bar and from the menu of options that appears, select Submit Job. Lastly, when the statement “Successfully Scheduled Network” appears in the Error frame, select Execution from the task menu bar and Start Simulation from the menu of options displayed. 

 

 

 

 

 

 

Viewing Results

When the statement “Network execution complete” appears in the Error frame, the results of any module can be displayed. To view a module’s results, right click on the module and a menu of options will appear.  Select Show Module Result from the menu of options.  The Result Dialog window will then be displayed.
